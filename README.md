# Casino Games

A fun collection of a variety of casino games, all coded in Python. Made for Horizons Crux Hackclub 2026.

To play, simply run any file in the terminal by double clicking the executable (the executables will be a universal Mac native file, both Silicon and Intel are able to use it). Money is stored in the "money.txt" file, so make sure that file is in the same folder as the game you want to play, so the code can access it.

Rules for all of the games comes from their corresponding Wikipedia page. There may be some discrepancies and oversimplifications from real life, but as an under 18 I cannot go to real casinos to fact-check.

Some sections of the code may have been inspired by responses from LLMs, but all code is ultimately written and debugged by me.
